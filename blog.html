DOM stands of Document Object Model. It is a tree-like structure that represents the HTML document. It is used by JavaScript to access and manipulate the document.
it's makes the html document interactive and html document manipulation and maintenance.

There are 5 ways in select HTML Elements.
  • getElementById()
  • getElementsByTagName()
  • getElementsByClassName()
  • querySelector()
  • querySelectorAll()
