DOM stands of Document Object Model. It is webpage programming interface and a tree-like structure that represents the HTML document. It is used by JavaScript to access and manipulate the document.
it's makes the html document interactive and html document manipulation and maintenance.

There are 5 ways in select HTML Elements.
  • getElementById()
  • getElementsByTagName()
  • getElementsByClassName()
  • querySelector()
  • querySelectorAll()

HTML contains different types of elements that are placed one after the other inside the parent tag. When an event occurs, the event tag rises above the parent tag and the parent tag above the parent tag rises slowly like a bubble. This event is called Event Bubble.

When a function is passed or used as an argument to another function, it is called a callback function. Since JavaScript function is basically an object, we can use functions like objects. Each can be called by declaring one inside the other.
